Comment lancer le programme ?

Ouvrir "index.html" dans le navigateur de votre choix.
L'application a été testée et a fonctionné sur les navigateurs suivants :
	- Brave
	- Google Chrome
	- Edge
	- Firefox

********************
Structure de l'application

Voici l'arborescence des fichiers du programme :
	| images/
	----| Blue.png : image du bonbon 5
	----| Green.png : image du bonbon 4
	----| Orange.png : image du bonbon 3
	----| Red.png : image du bonbon 2
	----| Yellow : image du bonbon 1
	| js/
	----| Bonbon.js : classe Bonbon
	----| Controleur.js : classe Controleur
	----| index.js : fichier initiant l'application
	----| Modele.js : classe Modele
	----| Vue.js : classe Vue
	| index.html : page html contenant le canvas du jeu.
